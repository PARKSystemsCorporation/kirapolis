# Presidential Memory Template

This is a starter dataset for testing the weight-unlearning pipeline.

## Goal

- Forget: direct recall of selected pre-2020 historical facts
- Retain: general software engineering and operational writing ability

## Files

- `forget.train.jsonl`
- `retain.train.jsonl`
- `forget.eval.jsonl`
- `retain.eval.jsonl`
- `general.eval.jsonl`

Edit these rows to match the exact memory or concept you want to weaken while preserving the rest of the model.
